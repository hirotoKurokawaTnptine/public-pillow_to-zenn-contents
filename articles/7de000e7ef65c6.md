## はじめに
タイトル通り  
本記事では、PowershellだとPSCustomObjectの影に隠れがちなクラス機能にスポットを当て、  
スクリプトでも扱いやすい、軽量な使い方を模索していく試みです。

## クラス vs カスタムオブジェクト
Powershellでは、主にカスタムオブジェクトの方が好まれているように感じます。
Powershellは、どちらかというと関数型パラダイムに寄っていて、オブジェクトはデータとしての役割が大きく、パイプラインから送られたオブジェクトを動的に加工できるカスタムオブジェクトは、クラスに比べ柔軟な操作が可能です。


## キャストによるインスタンス化
クラスのインスタンス化の方法はいくつかありますが、今回はハッシュテーブルを利用した型キャストを利用します。
```Powershell
class Hoge {
    [string]$foo
}

$hoge = [Hoge]@{
    foo = "Foo"
}

$hoge.foo # Fooと表示される

```
カスタムオブジェクトを定義する時と同じですね。
